# Landing Page
The Odin Project Foundations' Curriculum: [Project: Landing Page](https://www.theodinproject.com/lessons/foundations-landing-page)

The goal of this project is to create an entire webpage from a design. The design includes what the entire website will look like and the fonts and colors used.

## Technologies
- HTML and CSS (Flexbox is used primarly to layout the content on the webpage)
